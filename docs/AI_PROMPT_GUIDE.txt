🎯 1. PROMPT CHO TÍNH NĂNG UI/FRONTEND
================================================================================

Template:
---------
"Tôi muốn thêm tính năng [TÊN TÍNH NĂNG] vào giao diện:
- Mô tả: [MÔ TẢ CHI TIẾT]
- Vị trí: [VỊ TRÍ HIỂN THỊ]
- Chức năng: [NHỮNG GÌ NGƯỜI DÙNG CÓ THỂ LÀM]

Hãy cập nhật file templates/main_spa.html và static/style.css"

Ví dụ cụ thể:
-------------
"Tôi muốn thêm tính năng Dark Mode vào giao diện:
- Mô tả: Toggle switch để chuyển đổi giao diện sáng/tối
- Vị trí: Góc trên bên phải header
- Chức năng: Click để chuyển đổi theme, lưu preference vào localStorage

Hãy cập nhật file templates/main_spa.html và static/style.css"

🔧 2. PROMPT CHO TÍNH NĂNG BACKEND/API
================================================================================

Template:
---------
"Tôi muốn thêm API endpoint mới:
- Endpoint: [ĐƯỜNG DẪN API]
- Method: [GET/POST/PUT/DELETE]
- Chức năng: [MÔ TẢ CHỨC NĂNG]
- Input: [DỮ LIỆU ĐẦU VÀO]
- Output: [DỮ LIỆU TRẢ VỀ]

Hãy cập nhật file src/app_restructured.py"

Ví dụ cụ thể:
-------------
"Tôi muốn thêm API endpoint mới:
- Endpoint: /api/search_memories
- Method: POST
- Chức năng: Tìm kiếm memories theo từ khóa
- Input: {"keyword": "text", "user": "username"}
- Output: {"results": [...], "count": number}

Hãy cập nhật file src/app_restructured.py"

💡 VÍ DỤ PROMPT HOÀN CHỈNH
================================================================================

Ví dụ 1 - Tính năng Share Memory:
---------------------------------
"Tôi muốn thêm tính năng Share Memory:

🎯 Mô tả: Cho phép user tạo link public để share memory với người khác
📍 Vị trí: Nút 'Share' trong lightbox của mỗi memory
🔧 Chức năng: 
- Click Share → tạo unique link
- Link có thể xem memory mà không cần đăng nhập
- Có thể set expiry date cho link
- Track số lượt xem

📝 Files cần update:
- templates/main_spa.html (thêm nút Share)
- src/app_restructured.py (API tạo share link)
- static/style.css (styling cho share modal)

Hãy implement tính năng này."

Ví dụ 2 - Tính năng Comment System:
-----------------------------------
"Tôi muốn thêm tính năng Comment cho memories:

🎯 Mô tả: Cho phép user comment vào memory của mình hoặc người khác
📍 Vị trí: Dưới mỗi memory trong lightbox
🔧 Chức năng:
- Hiển thị danh sách comments
- Thêm comment mới
- Xóa comment của mình
- Realtime update (optional)

📝 Files cần update:
- templates/main_spa.html (UI comments)
- src/app_restructured.py (API comments)
- static/style.css (styling comments)
- LuuTru_User/{user}/comments.json (data storage)

Hãy implement tính năng này."

Ví dụ 3 - Tính năng Export Data:
--------------------------------
"Tôi muốn thêm tính năng Export dữ liệu user:

🎯 Mô tả: Cho phép user export toàn bộ dữ liệu của mình
📍 Vị trí: Settings page
🔧 Chức năng:
- Export thành ZIP file
- Bao gồm: images, videos, memories, comments
- Có thể chọn date range
- Email link download

📝 Files cần update:
- templates/main_spa.html (settings page)
- src/app_restructured.py (export API)
- scripts/export_utility.py (export logic)
- config/email_config.py (email notification)

Hãy implement tính năng này."
